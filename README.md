# 💰 Expense Tracker Android App

An Android application built with **Kotlin and Jetpack Compose** for tracking personal expenses, monitoring budgets, and visualising monthly spending patterns.

## 🚀 Features

- ➕ Add, edit, and delete expenses instantly
- 📅 Create and manage expense sheets by month and year
- 📊 Interactive charts — drag to view spending across different months
- 🗄️ Local database storage using SQLite — works fully offline
- 🧾 View all expenses per month in a clean list layout
- ✏️ Edit and update existing expense entries in real time

---

## 🛠️ Tech Stack

| Technology | Purpose |
|---|---|
| Kotlin | Primary programming language |
| Jetpack Compose | Modern declarative UI framework |
| SQLite | Local database for persistent storage |
| Android SDK | Core Android development framework |
| Gradle | Build system |

## ⚙️ How to Run

1. Clone this repository
```bash
git clone https://github.com/koushil001/expense-tracker-android.git
```

2. Open the project in **Android Studio**

3. Wait for Gradle to sync all dependencies

4. Run the app on an emulator or physical Android device

> Minimum SDK: Android 8.0 (API level 26)
